# 🍣 Sushi Counter (寿司计数器) v1.0

## 一个“无用但有趣”的计数工具

### 💡 灵感来源
你有去过回转寿司店吗？
看着服务员阿姨数着桌上不同颜色的盘子：红盘 6 元，黄盘 12 元，黑盘 28 元... 啪啪啪按几下计算器，账单就出来了。

那一刻我就在想：**要是有一个专门数盘子的 APP 就好了！**

于是，这个 **Sushi Counter** 诞生了。
虽然它现在长得像个通用计数器（可以算 AA 账、购物清单），但它的灵魂永远属于那堆叠得高高的寿司盘。

### ✨ 主要功能
*   **多行计数**：项目、单价、数量，自动算小计。
*   **实时总计**：底部永远显示你要付多少钱。
*   **换肤系统 (核心卖点)**：
    *   🔘 **iOS 6**：拟物化时代的眼泪（亚麻布、皮革、阴影）。
    *   ⚪️ **iOS 7**：扁平化时代的开始（极简、纯白）。
    *   🔮 **iOS 26**：来自未来的脑洞（液态玻璃、霓虹光晕）。
*   **数据持久化**：不小心刷新页面？没关系，数据还在（Local Storage）。
