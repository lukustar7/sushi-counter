# 寿司计数器 v3.0.2

纯前端寿司点单计数器，用于记录经典盘数量、自定义餐品、茶位费、就餐人数，并实时计算总额与人均费用。

## 功能

- 经典五色盘支持数量加减与单价编辑，数据自动保存在浏览器本地。
- 自定义餐品支持名称、单价、数量录入，并与经典盘和茶位费合并结算。
- 支持 PWA 安装与离线使用，适合手机主屏幕快捷打开。

## 运行

项目无构建步骤、无第三方依赖。

```bash
python3 -m http.server 8765
```

然后打开：

```text
http://localhost:8765/
```

直接打开 `index.html` 可以使用基础计数功能，但 Service Worker 与 PWA 离线缓存需要在 `localhost` 或 HTTPS 静态服务下运行。

## 数据存储

数据保存在浏览器 `localStorage` 中，键名均以 `sushi_` 开头。重置操作只清理本应用数据，不影响同源下其它项目的本地数据。
