# 🍣 寿司郎专属计数器 (Sushiro Board) v2.0

## 🎯 专为“寿司郎高频点餐”而生的专业收银账单面板

回转寿司的精髓在于那高高叠起的彩色盘子。本工具专为寿司郎门店量身定制，通过拟真图形化彩盘网格与智能化就餐结算，助您和朋友在吃寿司时，一秒算清账单与人均费用！

---

## ✨ 核心大招与升级特性 (v2.0)

1. **🍅 寿司郎正宗视觉品牌锁死**
   - 彻底删除旧版复杂的换肤切换逻辑。
   - 采用正宗**“寿司郎大红”**与日式原木**“暖米白”**作为全局唯一的视觉主题，搭配页头**旋转传送带动态线**，食欲与高级感直接拉满！

2. **🎨 立体拟物“双行五色圆形寿司盘”快捷点按区**
   - 经典寿司盘升级至 **5 款**，完美对应门店真实的盘色与价格：
     - **第一行 (高频款)**：🥚 **白盘 (8元)**、🔴 **红盘 (10元)**
     - **第二行 (豪华款)**：⚪ **银盘 (15元)**、🟡 **金盘 (20元)**、⚫ **黑盘 (28元)**
   - **“把盘子画出来！”**：用纯 CSS 高级偏光阴影，绘制出富有 3D 浮雕拟物感的圆形寿司盘。**点击圆盘任何位置即可快速 +1 计数**，超级爽快！
   - 每个盘子下方自带大触控的独立 `+` 和 `-` 微调按钮，防误触体验极佳。

3. **🍵 智能化就餐配置（茶位费 + 人数）**
   - 页面顶部设有精致的就餐管理卡片：
     - **茶位单价**：默认 5 元/人，可点按直接修改。
     - **就餐人数**：默认 1 人，支持加减或直接输入。
   - 茶位费（`单价 × 人数`）自动计算并实时计入总账单。

4. **💵 实时账单与“人均费用”自动结算**
   - 底部特化为**账单小票样式**的控制面板。
   - **就餐人均费用**实时计算并加粗醒目显示。
   - 具备严密的防呆设计，当人数减少至 0 时人均自动等于总计，绝不报错或显示 `NaN`。

5. **🍜 横行自定义单点餐品区**
   - 对于非标准盘子的单点美食（例如拉面、天妇罗、甜品等），点击底部的巨大 `+` 按钮即可随心添加。
   - 完美沿用原先经典的**“横行自适应卡片设计”**，保持界面简洁整齐。

6. **🔋 物理按压微动效与 LocalStorage 持久化**
   - 所有的按钮（加减、重置、添加）均附带** `:active` 物理缩放回弹微动效**，触控质感犹如实体点单机。
   - 无论页面刷新、手滑退出，就餐人数、经典盘数量和自定义单点行数据均自动保存在浏览器本地，绝对防丢。
