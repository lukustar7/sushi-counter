# CHANGELOG (项目更新日志)

本项目的所有重大变更、功能新增或 Bug 修复都将记录在此文件中。

---

## [2.0.0] - 2026-06-01

### 🍣 终极重构与全新设计 (Major Release)
- **正宗「寿司郎收银面板」**：
  - 移除了旧版的 iOS 6, iOS 7, iOS 26 换肤功能，将极具品牌感的**寿司郎原木大红**（`#E60012` + `#FAF8F3`）设为网页唯一的、专属的主视觉风格。
  - 页头标题居中放大，整体更具商用结算终端的高级质感。
- **「双行五色拟物圆形寿司盘」网格点按区**：
  - 将默认的经典寿司盘升级为 **5 款彩盘**，新增了 **🥚 白盘 (8元)**！
  - 进行了重大的交互升级，将这 5 款盘子从纵向列表中剥离，提炼为顶部独立的 **点按式圆形网格卡片**，分两行排版：
    - **第一行 (2款)**：🥚 白盘 (8元)、🔴 红盘 (10元)
    - **第二行 (3款)**：⚪ 银盘 (15元)、🟡 金盘 (20元)、⚫ 黑盘 (28元)
  - 使用纯 CSS 精心渲染了**极富 3D 浮雕拟物立体感的圆形寿司盘**，点击盘子任意位置即可快速完成该色盘的 `+1` 计数！
  - 盘子下方配有大而平滑的 `+` 与 `-` 独立微调按钮，在 0 盘时减号自动淡化。
- **🍵 茶位配置面板**：
  - 新增就餐基本配置卡片，支持设置**茶位费单价**（默认 5 元）和**就餐人数**（默认 1 人，带大加减按键）。
  - 茶位费自动计算 `茶位费单价 × 就餐人数`，并实时累入总额。
- **👥 人均消费自动结算**：
  - 底部账单面板新增**人均消费（Average）**实时展示。
  - 设计了严密的防除零呆滞逻辑，如果人数修改为 0 人，人均会自动以总额显示，保障计算健壮。
- **自定义单点餐品行**：
  - 原本的纵向列表区升级为独立的“自定义单点餐品区”，保留经典好看的横行输入框设计，专门用于点加号自由录入其他拉面、甜品、饮料等小食。

## [1.1.0] - 2026-06-01

### 🍣 新增功能 (Added)
- **「寿司郎 (Sushiro) 特供版」全新主题**：
  - 深度致敬寿司郎官方视觉设计语言。
  - 使用暖木浅米色背景 (`#FAF8F3`) 和高饱和度寿司郎标志红 (`#E60012`)，打造沉浸式用餐氛围。
  - 新增**旋转传送带动态装饰线条**（红黄相间条纹平移微动画）。
  - 新增底部**账单小票质感栏**与大红圆形弹性计数按钮。
- **全局物理交互微动效**：
  - 增加所有点击按钮的按压物理缩放弹性反馈 (`scale(0.9)`)，提升单手触屏操作流畅度。
- **寿司郎经典四色盘初始计价矩阵**：
  - 将默认的初始盘子数据全面升级为真实的寿司郎彩盘规格：红盘 (10元)、银盘 (15元)、金盘 (20元)、黑盘 (28元)。

### 🎨 界面优化 (Improved)
- 优化了全局新增行的随机配色逻辑，引入精心配置的“和谐寿司色板”（不再生成刺眼或低对比度色彩）。
